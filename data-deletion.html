Suppression de vos données

Cette page explique comment demander la suppression de votre compte et des données associées dans l’application Flagged.

1. Suppression via l’application

Lorsque l’option sera disponible, vous pourrez supprimer votre compte directement depuis les paramètres de l’application (profil / compte / suppression). La suppression entraînera la suppression de vos données de compte et, autant que possible, des contenus associés, sauf obligations légales contraires.

2. Demande de suppression par e-mail

Si vous ne trouvez pas encore l’option dans l’application, vous pouvez demander la suppression manuelle de vos données en nous écrivant à :

  • Adresse : moreno.cle@gmail.com
  • Objet recommandé : « Suppression de mon compte Flagged »
  • Contenu : indiquez l’adresse e-mail utilisée pour votre compte et précisez que vous souhaitez la suppression définitive de votre compte et de vos données.

Nous traiterons votre demande dans un délai raisonnable, sauf si une conservation est nécessaire (obligations légales, prévention de fraudes, litiges).

3. Données pouvant être conservées temporairement

Certaines données techniques (journaux de connexion, informations de sécurité) peuvent être conservées pour une durée limitée afin de :

  • respecter des obligations légales ;
  • assurer la sécurité du service ;
  • prévenir la fraude ou l’abus ;
  • gérer d’éventuels litiges.

4. Contact

Pour toute question sur la suppression de vos données, écrivez à : moreno.cle@gmail.com
